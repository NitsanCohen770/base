## CSS Component for passing the color-palette as CSS-Variables

### Colors Available:

<div className={colorPalette.color}>
  <div style={{ backgroundColor: 'var(--text-gray)' }}> Gray</div>
  <div style={{ backgroundColor: 'white', color: 'var(--text-black)' }}>
    Black Text
  </div>
  <div style={{ backgroundColor: 'black', color: 'var(--text-white)' }}>
    White Text
  </div>
  <div style={{ backgroundColor: 'green', color: 'var(--prev-price)' }}>
    {' '}
    This one is used for previous prices:<span
      style={{ textDecoration: 'line-through' }}
    >
      {' '}
      99 $
    </span>
  </div>
</div>
